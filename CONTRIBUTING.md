# Contributing to pyenv-doctor

Thank you for your interest in improving `pyenv-doctor`.

This is a small, beginner-friendly open source project. Contributions are welcome, especially bug reports, documentation improvements, small fixes, and focused feature suggestions.

## How to Run the Project Locally

Clone the repository and run the project from the repository root:

```powershell
python main.py
```

You can also scan a specific directory:

```powershell
python main.py .
python main.py "C:\path\to\project"
```

## Reporting Bugs

If you find a bug, please open an issue and include:

- your operating system
- your Python version
- the command you ran
- the actual output
- the expected result
- any extra context that may help reproduce the problem

Please use the bug report template when possible.

## Suggesting Features

Feature requests are welcome.

When opening a feature request, please explain:

- what problem you want to solve
- your proposed idea
- the use case
- any alternatives you considered

Please use the feature request template when possible.

## Pull Requests

Pull requests are welcome for small, focused improvements.

Before opening a PR, please make sure that:

- the code runs successfully
- the output matches the current README
- relevant documentation has been updated when needed
- the change is limited in scope and easy to review

## Keep It Simple

Please keep changes beginner-friendly and avoid unnecessary complexity.

Thank you for helping improve `pyenv-doctor`.
